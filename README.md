# EDA-Tool
Hello. Are you tired of running code for general data exploration again and again. Fear not, I am here with a EDA-Tool for general data exploration.

## How to use
```
python dashboard.py <csv file>
```

## Current capability
* only read csv file
* following graphs:
  1. Line Graph
  2. Scatter Plot
  3. Histogram
  4. Density Graph
  5. Correlation heatmap
  6. Pie Chart
  7. Bai Chart
  8. Violin Graph
  9. Box Chart
  10. Count heatmap
